# Example (Vite)

This is the demo site built for GitHub Pages.

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```

Notes:
- `example/vite.config.ts` uses `base: /react-typewriter-hook/` in production to match GitHub Pages under the repo name.
